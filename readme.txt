It's a Sexy and Nice plugin to get reported your site's Problem/Broken Links

== Description ==

This plugin simply adds a button to Your site and let users report Your site broken links/problem and You can see them in WP dashboard.
Description Box let You get site visitors name, email or anything You need plus description of what is the problem but if You don't need this feature simply You can disable It from setting page
With shortcode provided You can use plugin fully customized :)

== Installation ==

1. Upload Zip file content to "/wp-content/plugins/" directory and extract it
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place "<?php RBL_UI(); ?>" in your template's Single.php file where you want
4. Or Create Your button and set "RBL_Element" as Its id
5. Or as an Alternate You Can Use RBL Shortcode: [RBL url="Link to Be Reported"]Caption of Button[/RBL]
